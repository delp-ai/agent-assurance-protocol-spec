# AAP Conformance Negative Cases

This artifact defines the negative gating corpus for the revised `AAP` base profile.

Its purpose is to force independent implementations to reject the same invalid behaviors
for the same machine-checkable reasons.

Like the positive corpus, these are canonical object-form cases paired with the byte corpus and
byte transcripts where exact protected frame bytes are required.

## Case 1: Invalid Bootstrap Version

Case id:

- `wire.bootstrap.invalid_version`

Input:

```text
Frame {
  Version: 1.0
  StreamId: 0
  MessageId: 1
  FrameType: HELLO
  Payload: HelloBody {
    peerAgentId: agent.alpha
    supportedMajorVersions: [1]
    supportedMinorRanges: [{minInclusive: 0, maxInclusive: 0}]
    resumeTokenPresent: false
    identityManifestRef: manifest.alpha.identity
    identityManifestInline: embedded(AgentIdentityAttestation(agent.alpha))
    bootstrapTranscriptDigest: sha256:boot_bad_version
    trustPolicyDigest: sha256:trust_policy_open_core_0001
    trustPolicyRef: manifest.trust.open_core
  }
}
```

Expected outcome:

```text
NegativeOutcome {
  accepted: false
  errorCode: malformedHeader
  errorScope: session
  retryable: false
  semanticEffect: none
}
```

Reason:

- pre-negotiation `HELLO` must use bootstrap `Version=0.0`

- `contract.dispute.budget_depleted`

```text
  contractRevision: 1
  subjectMessageRef: {streamId: 4, messageId: 2}
  subjectDigest: sha256:evidence_exec_0001b

  requestedRepair: attachMissingProof
  deadlineUnixMs: 1773034900000
  disputeDepth: 2
  challengeCostRef: dispute.compute.0004
  authorityBasisDigest: sha256:graph_exec_0001
  canonicalDigest: sha256:challenge_exec_0004
```

```text
NegativeOutcome {
  accepted: false
  contractStateRemains: Proved
```

Reason:

- an admissible challenge path ends when the accepted dispute budget is exhausted; further
  challenges must not be admitted on local goodwill alone

## Negative Corpus Rule

These cases define the shared rejection semantics for the gating corpus.

Each case should be interpreted together with the byte corpus and byte transcripts when the
offending input is transported as a protected AAP frame.

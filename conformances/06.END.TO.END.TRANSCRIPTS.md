# AAP Conformance End-To-End Transcripts

This artifact publishes composed reference trace indexes from bootstrap through terminal close.

These transcripts are normative for lineage binding and required terminal properties, but they are
not standalone byte-or-semantic oracles: independent implementations still consume the referenced
bootstrap, contract, dispute, resume, and byte-corpus artifacts to reconstruct the exact ordered
frame and object lineage.

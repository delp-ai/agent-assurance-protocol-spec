# AAP Conformance Negotiation And Race Cases

This artifact extends the gating corpus with deterministic negotiation, supersession, timer, and
race outcomes that independent implementations must reproduce without human interpretation.

## Case 1: `COUNTER` Supersedes Active Proposal

```text
Precondition {
  activeCandidateDigest: sha256:proposal_exec_r1
  activeCandidateRevision: 1
  contractRef: {issuerAgentId: agent.alpha, contractId: contract.exec.0001}
}

Frame {
  Version: 1.0
  StreamId: 2
  MessageId: 2
  FrameType: COUNTER
  ContractRef {
    issuerAgentId: agent.alpha
    contractId: contract.exec.0001
    contractRevision: 2
    contractPhase: countered
  }
  Payload: CounterBody {
    contractId: contract.exec.0001
    priorRevision: 1
    counterRevision: 2
    supersededProposalDigest: sha256:proposal_exec_r1
    contractDigest: sha256:contract_exec_r2
    contractSchemaRef: manifest.contract.exec.0002
    authorityGraphDigest: sha256:graph_exec_r2
    counterDigest: sha256:counter_exec_r2
    counteringAgentId: agent.beta
    counteredAtUnixMs: 1773033602000
    counterSignature: sig.counter.beta.r2
  }
}

CounterOutcome {
  accepted: true
  resultingState: Countered
  activeCandidateDigest: sha256:counter_exec_r2
  staleCandidateDigests: [sha256:proposal_exec_r1]
}
```
